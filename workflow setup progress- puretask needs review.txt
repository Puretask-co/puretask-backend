The GPT tool function update_node_parameters can only write:

✔ strings
✔ numbers
✔ booleans
✔ simple objects

{
  "__rl": true,
  "value": "workflowIdString",
  "mode": "list",

  "cachedResultName": "04 – Processor Name"
}
The n8n tool API cannot generate the internal metadata, so it collapses back to placeholder.
