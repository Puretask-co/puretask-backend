## 🎯 **Template Types & When to Use:**

```
BOOKING STAGE        →    TEMPLATE TYPE
─────────────────────────────────────────

New Booking          →    Booking Confirmation
                          "Hi {client_name}! Confirmed for {date}..."

Before Appointment   →    Reminder
                          "Hi {client_name}! Just a friendly reminder..."

During Service       →    Running Late
                          "Hi {client_name}, running 15 min late..."

After Service        →    Job Complete
                          "Hi {client_name}! All done! ✨"

Follow-up            →    Review Request
                          "Would you mind leaving a review? 🙏"

Rebooking            →    Special Offer
                          "Book again this month for 10% off!"
```
