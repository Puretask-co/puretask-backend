## 🎚️ Automation Levels

| Mode | What It Does | Best For |
|------|--------------|----------|
| **Manual** ✋ | You do everything | New users (Week 1-2) |
| **Assisted** 🤝 | AI suggests, you approve | Most users (Recommended) |
| **Full Auto** 🚀 | AI does everything | Experienced users |

**Current recommendation:** Assisted Mode for first 4-6 weeks

## 🚨 Emergency Actions

**If AI sends wrong message:**
1. Send personal correction immediately
2. Apologize to client
3. Switch to Manual/Assisted Mode
4. Fix the template
5. Report the issue

**If unsure about AI suggestion:**
- In Assisted Mode: Don't approve, write manually
- Better safe than sorry!

**To pause AI completely:**
- Settings → Emergency Stop
- Or switch to Manual Mode

## 🔧 Troubleshooting

| Problem | Quick Fix |
|---------|-----------|
| AI sounds too formal | Lower formality level to 2-3 |
| AI sounds too casual | Raise formality level to 3-4 |
| Too many emojis | Change to "Minimal" or "Moderate" |
| Templates don't sound like me | Go edit them! |
| Not saving time | Enable more automation gradually |
| Worried about mistakes | Stay in Assisted Mode longer |
