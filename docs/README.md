# Docs and Historical Artifacts

This folder houses reference material plus historical AI-generated analysis, postmortems, and migration notes used during stabilization of the PureTask backend. None of these files are required at runtime; they are here for context and audit history.

## What to expect
- Reference docs: API, deployment, environment templates, and testing guides.
- Historical artifacts: analysis reports, best-practice reviews, migration notes, and checklists moved from the repo root.

## Contributing to docs
- Keep runtime/config docs up to date (API, env templates, deployment).
- When adding investigative write-ups or postmortems, place them here to keep the root clean.
- Do not store secrets in this folder.
